# Skills for engineers

[![skills.sh](https://skills.sh/b/Oudwins/skills)](https://skills.sh/Oudwins/skills)

Inspired by [matt pocock's brilliant skills](https://github.com/mattpocock/skills/tree/main) I have decided to start compiling my own personal agent skills that I use every day to do real engineering work.

I have never found much success with his system for PDRs or other structured approaches like GSD, BMAD or Spec-Kit. So this is focused on powerful skills that accelerate human in the loop work.

## Install

Install with [`skills.sh`](https://github.com/vercel-labs/skills):

```bash
npx skills@latest add Oudwins/skills
```
